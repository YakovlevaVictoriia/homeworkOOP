# Проект ООП для составления списка продуктов
## Описание программы

С помощью данного кода можно:
- Создавать ингредиенты с указанием названия, количества и единиц измерения
- Создавать рецепты блюд из ингредиентов
- Увеличивать рецепты на нужное количество порций
- Формировать список продуктов из нескольких рецептов
- Использовать список продуктов максимально эффективно - суммировать нужные продукты из разных блюд
- Добавлять диетические рецепты

## Использование кода

Развернуть проект можно так: 
```bash
git clone <ваш-репозиторий>
cd <папка>
pip install -r requirements.txt
pytest
```

### Как запустить работу тестов 

Для начала нужно перейти в папку с проектом.

`cd путь к вашему проекту`

Далее можно запустить все тесты

`pytest test_recipes.py`

Или какие-то определенные тесты:

`pytest test_recipes.py::TestIngredient::test_ingredient`

### Как проверить использование программы

Создайте экземпляры каждого класса, попробуйте использовать различные методы, записанные в коде программы.

Запуск кода main в терминале:

`python main.py`
